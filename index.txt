                <h2>📚 Economics Adventure</h2>
                <div class="card card-gold">
                    <div class="card-title">🌟 Unit: Maykroekonomiks</div>
                    <p>A journey through the secrets of economics. Two visual lessons per topic.</p>
                </div>
                <div class="card">
                    <div class="card-title">1️⃣ Demand</div>
                    <p><strong>📘 Kahulugan:</strong> Ang demand ay tumutukoy sa dami ng produkto o serbisyo na gusto at kayang bilhin ng mamimili sa isang takdang presyo at panahon.</p>
                    <p><strong>Batas ng Demand: Ceteris paribus</strong> — Kapag tumataas ang presyo, bumababa ang demand; kapag bumababa ang presyo, tumataas ang demand.</p>
                    <p><strong>Mga Salik na Nakaaapekto sa Demand:</strong> Kita ng Mamimili, Panlasa at Kagustuhan, Presyo ng Kaugnay na Produkto, Dami ng Mamimili, Inaasahan sa Hinaharap.</p>
                    <div class="media-container">
                        <img src="s&d.jpg" alt="Demand (2)">
                        <img src="law.png" alt="Demand (1)">
                    </div>
                </div>
                <div class="card">
                    <div class="card-title">2️⃣ Supply</div>
                    <p><strong>📘 Kahulugan:</strong> Ang supply ay dami ng produkto o serbisyo na handa at kayang ipagbili ng mga prodyuser sa takdang presyo at panahon.</p>
                    <p><strong>Batas ng Supply: Ceteris paribus</strong> — Kapag tumataas ang presyo, tumataas ang supply; direktang ugnayan sa pagitan ng presyo at dami ng supply.</p>
                    <p><strong>Mga Salik:</strong> Teknolohiya, Presyo ng Input, Bilang ng Prodyuser, Inaasahan sa Presyo, Kalikasan at Panahon.</p>
                    <div class="media-container">
                        <img src="ply.png" alt="Supply (2)">
                        <img src="sup.jpg" alt="Supply (1)">
                    </div>
                </div>
                <div class="card">
                    <div class="card-title">3️⃣ Elasticity</div>
                    <p><strong>📘 Kahulugan:</strong> Ang price elasticity ay sumusukat kung gaano kalaki ang pagtugon ng demand o supply sa pagbabago ng presyo.</p>
                    <p><strong>Mga Uri:</strong> Elastic Demand, Inelastic Demand, Unitary Elastic.</p>
                    <p><strong>Halimbawa:</strong> Kapag tumaas ng 10% ang presyo ng softdrinks at bumaba ang demand ng 30%, ito ay elastic.</p>
                    <div class="media-container">
                        <img src="elasticity.jpg" alt="Types of Elasticity">
                        <img src="Price_Elasticity.png" alt="Types of Elasticity (2)">
                    </div>
                </div>
                <div class="card">
                    <div class="card-title">4️⃣ Market Equilibrium</div>
                    <p><strong>📘 Kahulugan:</strong> Ang market equilibrium ay kalagayan kung saan nagtatagpo ang demand at supply — walang labis (surplus) o kulang (shortage).</p>
                    <p><strong>Mga Sangkap:</strong> Equilibrium Price at Equilibrium Quantity.</p>
                    <div class="media-container">
                        <img src="equilibrium-quantity.jpg" alt="Market Equilibrium">
                        <img src="elp.png" alt="Market Equilibrium (2)">
                    </div>
                </div>
                <div class="card">
                    <div class="card-title">5️⃣ Iba't Ibang Estruktura ng Pamilihan</div>
                    <p><strong>1. Ganap na Kompetisyon</strong> — Maraming nagbebenta at bumibili; pare-pareho ang produkto. Hal: palengke, gulayan.</p>
                    <p><strong>2. Monopolyo</strong> — Isang kompanya lang ang nagbebenta. Hal: Meralco.</p>
                    <p><strong>3. Monopolistikong Kompetisyon</strong> — Maraming nagbebenta pero may kaunting pagkakaiba sa produkto. Hal: sabon, toothpaste.</p>
                    <p><strong>4. Oligopolyo</strong> — Kaunting kompanya ang nagkokontrol ng presyo. Hal: telco companies, oil firms.</p>
                    <div class="media-container">
                        <img src="lesson5(1).png" alt="Estruktura ng Pamilihan">
                        <img src="estruktura.png" alt="Estruktura ng Pamilihan (2)">
                    </div>
                </div>
                <div class="card">
                    <div class="card-title">🌸 Aralin 6 — Bahaging Ginagampanan ng Pamahalaan sa Ekonomiya</div>
                    <p><strong>Layunin:</strong> Panatilihin ang patas na kompetisyon, protektahan ang mamimili at prodyuser, kontrolin ang presyo at iwasan ang pang-aabuso.</p>
                    <p><strong>🏷️ Price Act (RA 7581)</strong> — Pinoprotektahan ang mamimili laban sa sobrang taas ng presyo.</p>
                    <p><strong>🛍️ Consumer Act (RA 7394)</strong> — Pinangangalagaan ang karapatan ng mamimili.</p>
                    <p><strong>📉 Price Ceiling</strong> — Pinakamataas na presyo na puwedeng ipataw ng nagtitinda. Hal: presyo sa bigas.</p>
                    <p><strong>📈 Price Floor</strong> — Pinakamababang presyo na itinatalaga ng pamahalaan. Hal: minimum wage.</p>

                    <p><strong>What is JavaScript?</strong> JavaScript is the most popular and widely used scripting language on the internet. It is recognized and works in all major browsers. It was developed by <strong>Brendan Eich</strong> of Netscape originally under the name <em>Mocha</em>, then renamed to <em>LiveScript</em>, and finally became JavaScript. It was opened to Internet Explorer in <strong>1996</strong> and is a trademark of the <strong>Oracle Corporation</strong>, licensed to entities like Mozilla Foundation.</p>

                    <p><strong>Purpose:</strong> JavaScript is used to create dynamic, interactive user experiences — handling UI rendering (like React for Facebook), real-time updates, personalized feeds, video controls, and tracking user activity. It powers platforms like Google, YouTube, and Facebook.</p>

                    <p><strong>Static vs. Dynamic Websites:</strong> A <strong>static webpage</strong> (flat/stationary page) is delivered to the browser exactly as stored — example: a photographer's portfolio. A <strong>dynamic website</strong> displays different content based on user actions or preferences — example: social media platforms.</p>

                    <p><strong>Key Characteristics of JavaScript:</strong> It is a client-side, high-level scripting, interpreted, and object-oriented language. <em>Client-side</em> means codes are processed on the user's own computer. <em>High-level scripting</em> means code is written in words close to English. <em>Interpreted</em> means the source code is converted into machine code at runtime. It is also case-sensitive, and semi-colons at the end of each statement are not required.</p>

                    <p><strong>What can JavaScript do?</strong></p>
                    <p>● <strong>React to events</strong> — mouse clicks, pre-loading of webpages, keydowns, and more.<br>
                    ● <strong>Validate data</strong> — instead of server-side scripting, JS can validate form input to save processing time.<br>
                    ● <strong>Create cookies</strong> — used to save/retrieve information from a visitor's computer to monitor habits and preferences.<br>
                    ● <strong>Enhance a webpage</strong> — makes content more dynamic and interactive.</p>

                    <p><strong>Features of JavaScript:</strong> Supports all structured C-syntax; has dynamic typing (variable types can change at runtime); can run locally for faster interaction; detects user actions like mouse clicks; and can be combined with CSS to produce <strong>DHTML</strong> (Dynamic HTML) — enabling effects like message scrollers, mouse trails, and falling snow.</p>

                    <p><strong>Characteristics:</strong> Lightweight, inserted into HTML pages, supported by all modern browsers, easy to learn, quick to download, no extra tools needed, license-free, and has a thriving online developer community.</p>

                    <p><strong>Disadvantages:</strong> Some older browsers (PDA, mobile) don't support JS; secrets embedded in JS can be extracted; JS and DOM can be exploited by malicious scripts; source code is sent to the client and can be reverse engineered.</p>

                    <p><strong>How to start using JavaScript:</strong> The <code>&lt;script&gt;&lt;/script&gt;</code> tag is used to write JavaScript. The <code>type</code> attribute defines the scripting language. To display output on a webpage, the <code>document.write</code> command is used.</p>

                    <p><strong>Difference — HTML vs CSS vs JavaScript:</strong><br>
                    🔶 <strong>HTML</strong> — creates the structure (layout, content, building blocks)<br>
                    🔷 <strong>CSS</strong> — stylizes the website (look and feel, responsiveness)<br>
                    🟡 <strong>JavaScript</strong> — increases interactivity (complex functions, dynamic features)</p>

                    <p><strong>Pagsasaka:</strong> pagliit ng lupang sakahan, paggamit ng makabagong teknolohiya, kakulangan ng mga pasilidad at imprastraktura sa kabukiran, kakulangan ng suporta mula sa iba pang sektor, pagbibigay prayoridad sa sektor ng industriya, pagdagsa ng dayuhang kalakal, Climate Change.</p>
                    <p><strong>Pangingisda:</strong> mapanirang operasyon ng malalaking komersiyal na mangingisda, epekto ng polusyon sa pangisdaan, lumalaking populasyon ng bansa, kahirapan sa hanay ng mga mangingisda.</p>
                    <p><strong>Paghahayupan:</strong> pagkakaroon ng mga peste sa alagang hayop tulad ng baboy at manok.</p>
                    <p><strong>Paggugubat:</strong> mabilis na pagkaubos ng mga likas na yaman.</p></div>
                    <div class="card"><div class="card-title">📜 Mga Patakaran at Programa sa Agrikultura</div>
                    <p><strong>Bago Colonized:</strong> Ang mga sinaunang Pilipino ay nakakapag-may-ari ng lupa sa pamamagitan ng pagiging unang nagtayo ng bahay. Wala pang titulo/rights noon — ang pinagbabasehan lamang ay salita ng bawat isa.</p>
                    <p><strong>Panahon ng Kastila — Encomienda System:</strong> Lupa bilang gantimpala sa matapat na naglilingkod sa hari ng Espanya at mga tumulong sa pananakop.</p>
                    <p><strong>Panahon ng Amerikano:</strong> Binili ang 160,000 hectares ng lupa mula sa mga prayle para sa mga magsasaka.</p>
                    <p>○ <strong>Land Registration Act ng 1902</strong> — Sistemang Torrens; nagkaroon ng mga titulo ng lupa.</p>
                    <p>○ <strong>Public Land Act ng 1902</strong> — Pamamahagi ng lupaing pampubliko sa mga pamilyang nagbubungkal.</p>
                    <p><strong>Panahon ng Kasarinlan:</strong></p>
                    <p>○ <strong>B.R. Blg. 1160</strong> — Nagtatag sa NARRA (National Resettlement and Rehabilitation Administration).</p>
                    <p>○ <strong>B.R. Blg. 1190 ng 1954</strong> — Proteksyon laban sa mapang-abuso na may-ari ng lupa; laban sa Absentee Landlord.</p>
                    <p>○ <strong>Agricultural Land Reform Code of 1963</strong> — Nagpalaya sa mabigat na trabaho ng magsasaka; nilagdaan ni Diosdado Macapagal noong Agosto 8, 1963.</p>
                    <p>○ <strong>Atas ng Pangulo Blg. 2 ng 1972</strong> — Isailalim sa reporma sa lupa ang buong Pilipinas noong panahon ni Marcos.</p>
                    <p>○ <strong>Atas ng Pangulo Blg. 27</strong> — Bibilhin ng pamahalaan ang mga lupang sakahan na taniman ng palay/mais na lampas sa 7 ektaryang limitasyon at ipamamahagi sa mga magsasakang walang lupa.</p>
                    <p>○ <strong>B.R. Blg. 6657 ng 1988 (CARL)</strong> — "Comprehensive Agrarian Reform Law" na inaprubahan ni Corazon Aquino noong Hunyo 10, 1988. Ipinamamahagi ang lahat ng lupang agrikultural anuman ang tanim nito sa mga walang lupang magsasaka.</p></div>
                    <div class="card"><div class="card-title">🏠 Hindi Sakop ng CARP</div>
                    <p>Liwasan at parke, mga gubat at reforestation area, mga palaisdaan, tanggulang pambansa, paaralan, simbahan, sementeryo, templyo, watershed, atbp.</p></div>
                    <div class="card"><div class="card-title">🏗️ Mga Naisakatuparan ng CARP (2003)</div>
                    <p><strong>1.</strong> Pagtatayo ng bahay-ugnayan (bagsakan) para sa mga magsasaka.</p>
                    <p><strong>2.</strong> Pagtatayo ng gulayan para sa mga magsasaka.</p>
                    <p><strong>3.</strong> Pagsisiguro na ang mga anak ng mga magsasaka ay makapag-aral — itinatag ang Pangulong Diosdado Macapagal Agrarian Reform Scholarship Program.</p>
                    <p><strong>4.</strong> Kalahi agrarian reform zones.</p></div>
                    <div class="card"><div class="card-title">🐟 Iba pang Programa at Ahensya</div>
                    <p><strong>Pangingisda:</strong> Pagtatayo ng mga daungan; Philippine Fisheries Code of 1998; Fishery Research.</p>
                    <p><strong>Pagtotroso:</strong> CLASP, NIPAS (National Integrated Protected Areas System), Sustainable Forest Management Strategy.</p>
                    <p><strong>Landbank of the Philippines</strong> — Upang magkaroon ng kaagapay ang mga magsasaka sa pagpapaunlad ng kanilang kabuhayan.</p>
                    <p><strong>DA</strong> — Gumagabay sa mga magsasaka ukol sa makabagong teknolohiya. &nbsp;<strong>BFAR</strong> — Paunlarin ang pangingisda. &nbsp;<strong>BAI</strong> — Paghahayupan. &nbsp;<strong>ERDB</strong> — Pananaliksik sa ecosystem.</p></div>

                <div class="ap-lesson" id="ap-l3">
                    <div class="card card-gold"><div class="card-title">🏭 Lesson 3: Sektor ng Industriya</div>
                    <p>Tumutukoy sa pagproseso ng mga hilaw na materyal at paggawa ng mga produkto na kinakailangan para sa pampublikong pamilihan. Naglalayong maiproseso ang mga hilaw na materyal upang makabuo ng produkto na ginagamit ng tao.</p></div>
                    <div class="card"><div class="card-title">🔧 Sub-Sektor ng Industriya</div>
                    <p><strong>1. Pagmimina (Mining)</strong> — Pagkuha at pagproseso ng mga yamang mineral (metal, di-metal, o enerhiya) upang gawing tapos na produkto. Ang mismong produkto, hilaw man o naproseso, ay nagbibigay ng kita para sa bansa.</p>
                    <p><strong>2. Pagmamanupaktura (Manufacturing)</strong> — Paggawa ng mga produkto sa pamamagitan ng manual labor o ng mga makina. Nagkakaroon ng pisikal o kemikal na transpormasyon ang mga materyal sa pagbuo ng mga bagong produkto.</p>
                    <p><strong>3. Konstruksyon (Construction)</strong> — Pagtatayo ng mga gusali, estruktura, at iba pang land improvements.</p>
                    <p><strong>4. Utilities</strong> — Pagtugon ng mga pangunahing pangangailangan ng mga mamamayan tulad ng tubig, kuryente, at gas. Paglalatag ng mga imprastraktura at angkop na teknolohiya upang maihatid ang serbisyo.</p></div>
                    <div class="card"><div class="card-title">💡 Kahalagahan ng Sektor ng Industriya</div>
                    <p><strong>1.</strong> Tumatanggap ng mga produkto mula sa sektor ng agrikultura at ng serbisyo mula sa sektor ng paglilingkod.</p>
                    <p><strong>2.</strong> Nagkakaloob ng hanapbuhay sa maraming Pilipino. &nbsp;<strong>3.</strong> Nagpapasok ng dolyar sa bansa.</p>
                    <p><strong>4.</strong> Nakagagamit ng makabagong Teknolohiya.</p>
                    <p><strong>5.</strong> Gumagawa ng mga intermediate products — mga materyales o produkto na ginagamit sa paggawa ng iba pang produkto (hal. wheat → to make bread).</p></div>
                    <div class="card"><div class="card-title">🔗 Pagkakaugnay ng Agrikultura at Industriya</div>
                    <p>● Nagmumula sa agrikultura ang mga hilaw na sangkap na ginagamit sa pagbuo ng mga produkto ng industriya.</p>
                    <p>● Ang mga traktora, sasakyang pangingisda, at iba pang produkto ay mula sa industriya na ginagamit ng mga magsasaka upang magkaroon ng mataas na produksyon.</p>
                    <p>● Ang dami ng lakas-paggawa ay nagpapakita ng lubos na pagtutulungan sa mga sektor ng industriya at agrikultura.</p>
                    <p>● Ang agrikultura ang pinagmumulan ng pagkain ng mga industriya na siyang tumutugon sa mga pagkaing kinakailangan ng mga tao.</p>
                    <p>● Ang industriya ang nagbibigay ng malaking kontribusyon sa ekonomiya.</p>
                    <p>● Ang mga sangkap ay nagkakaroon ng transpormasyon dahil sa mga lakas-paggawa upang madagdagan ang halaga nito at lalo pang gumanda.</p></div>
                    <div class="card"><div class="card-title">⚠️ Suliranin ng Industriya</div>
                    <p><strong>1.</strong> Pondo at kompetisyon &nbsp; <strong>2.</strong> Paghina ng lokal na industriya &nbsp; <strong>3.</strong> Demand</p></div>
                    <div class="card"><div class="card-title">📜 Mga Patakarang Pang-Ekonomiya</div>
                    <p><strong>Filipino First Policy</strong> — Programang Filipino Muna ni Pangulong Carlos P. Garcia. Naglalayong bigyan ng higit na malaking partisipasyon ang mga Pilipino sa pagpapatakbo ng ekonomiya.</p>
                    <p><strong>Oil Deregulation Law (R.A. 8479)</strong> — Nagbukas ng kompetisyon sa industriya ng langis. Ang pamilihan ang nagdidikta sa presyo ng petrolyo.</p>
                    <p><strong>Patakaran ng Microfinancing</strong> — Serbisyong pinansiyal na magpautang na mayroong mababang interes sa mahihirap na pamilya.</p>
                    <p><strong>Patakaran sa Online Business</strong> — Online shopping o retailing, online intermediary service, online advertisement o classified ads, online auction.</p>
                    <p><strong>Retail Nationalization Act of 1964 (R.A. 1180)</strong> — Sampung taong palugit sa mga dayuhang capitalist. Pilipino na may korporasyon o asosasyon lamang ang nasa kalakalang pagtitingi.</p>
                    <p><strong>Retail Trade Liberalization Act of 2000 (R.A. 8762)</strong> — Nagpawalang-bisa sa RA 1180. Binuksan ang kalakalang pagtitingi sa mga dayuhan.</p>
                    <p><strong>Build-Operate-Transfer (BOT) (R.A. 6957)</strong> — Binibigyan ng karapatang bumuo ng mga pasilidad sa kanilang mga sariling pondo at palakarin ito sa humigit-kumulang na 25 taon.</p>
                    <p><strong>Asset Privatization Trust (APT)</strong> — Itinatag upang mapangasiwaan ang pagbebenta ng mga korporasyong pagmamay-ari ng pamahalaan sa pribadong pagmamay-ari. Pinasimula ni Corazon Aquino.</p></div>
                    <div class="card"><div class="card-title">🏛️ Sangay ng Pamahalaan na Tumutulong sa Industriya</div>
                    <p><strong>DTI</strong> — Tuwirang tumitingin at nangangalaga sa larangan ng industriya at pangangalakal.</p>
                    <p><strong>BOI</strong> — Tinutulungan ang mga nagsisimulang industriya at humihikayat sa mga dayuhang mamumuhunan.</p>
                    <p><strong>PEZA</strong> — Tumutulong sa mga namumuhunan na maghanap ng lugar upang pagtayuan ng negosyo.</p>
                    <p><strong>SEC</strong> — Nagtatala at nagrerehistro sa mga kompanya sa bansa.</p>
                    <p><em>Industriyalisasyon: paggamit ng mga makinarya at makabagong kagamitan sa paglinang ng mga industriya. Sekundaryang sektor ang kumakatawan sa industriya.</em></p></div>

                <div class="ap-lesson" id="ap-l4">
                    <div class="card card-gold"><div class="card-title">🛎️ Lesson 4: Sektor ng Paglilingkod</div>
                    <p>Ang umaalalay sa buong yugto ng produksyon, distribusyon, kalakalan, at pagkonsumo ng mga produkto sa loob o labas ng bansa. Kilala rin ito bilang <strong>tersarya o ikatlong sektor ng ekonomiya</strong> matapos ang agrikultura at industriya.</p></div>
                    <div class="card"><div class="card-title">📋 Pormal na Sektor sa Ilalim ng Paglilingkod</div>
                    <p><strong>● Transportasyon, Komunikasyon, at mga Imbakan</strong> — Mga serbisyong pampublikong sasakyan, mga paglilingkod ng telepono at mga pinapaupahang bodega.</p>
                    <p><strong>● Kalakalan</strong> — Pagpapalitan ng iba't-ibang produkto at serbisyo.</p>
                    <p><strong>● Pananalapi</strong> — Serbisyong may kinalaman sa bangko, insurance, at pamumuhunan. Nagbibigay ng tulong pinansyal gaya ng mga bangko, bahay-sanglaan, atbp.</p>
                    <p><strong>● Paupahang Bahay at Real Estate</strong> — Mga paupahan tulad ng mga apartment, subdivision developer, town house, at condominium.</p>
                    <p><strong>● Paglilingkod ng Pampribado</strong> — Lahat ng mga paglilingkod na nagmumula sa pribadong sektor.</p>
                    <p><strong>● Paglilingkod ng Pampubliko</strong> — Lahat ng paglilingkod na ipinagkakaloob ng pamahalaan.</p></div>
                    <div class="card"><div class="card-title">🏛️ Mga Ahensyang Tumutulong sa Sektor ng Paglilingkod</div>
                    <p><strong>DOLE</strong> — Pangangalaga sa kapakanan ng mga manggagawang Pilipino, pagtataguyod ng maayos na oportunidad sa trabaho.</p>
                    <p><strong>OWWA</strong> — Tumitingin sa kapakanan ng mga Overseas Filipino Workers.</p>
                    <p><strong>POEA</strong> — Isulong at paunlarin ang mga programa ukol sa paghahanapbuhay sa ibayong-dagat. Itinatag sa bisa ng Executive Order 797 (1982).</p>
                    <p><strong>TESDA</strong> — Hikayatin ang buong partisipasyon ng industriya, paggawa, at mga institusyong teknikal at bokasyonal upang sanayin at paunlarin ang kasanayan ng mga manggagawa. Itinatag sa bisa ng R.A. 7796 (1994).</p>
                    <p><strong>PRC</strong> — Nangangasiwa at sumusubaybay sa gawain ng mga manggagawang propesyonal upang matiyak ang kahusayan sa paghahatid ng mga serbisyong propesyonal sa bansa.</p>
                    <p><strong>CHED</strong> — Nangangasiwa sa gawain ng mga pamantasan at kolehiyo sa bansa upang maitaas ang kalidad ng edukasyon sa mataas na antas.</p></div>
                    <div class="card"><div class="card-title">⚖️ Mga Batas na Nangangalaga sa Karapatan ng Manggagawa</div>
                    <p><strong>Artikulo 94 — Holiday Pay</strong> — Bayad ng isang manggagawa na katumbas ng isang araw na sahod kahit hindi pumasok sa araw ng pista opisyal.</p>
                    <p><strong>R.A. 6727 — Wage Rationalization Act</strong> — Nagsaad ng mga mandato para sa pagsasaayos ng pinakamababang pasahod o minimum wage.</p>
                    <p><strong>Artikulo 91-93 — Premium Pay</strong> — Karagdagang bayad sa manggagawa sa loob ng walong oras na trabaho sa araw ng pahinga at special days.</p>
                    <p><strong>Artikulo 87 — Overtime Pay</strong> — Karagdagang bayad sa pagtatrabaho na lampas sa 8 oras sa isang araw.</p>
                    <p><strong>Artikulo 86 — Night Shift Differential</strong> — Karagdagang bayad sa pagtatrabaho sa gabi na hindi bababa sa 10% ng kanyang regular na sahod sa bawat oras na ipinagtrabaho sa pagitan ng 10PM–6AM.</p>
                    <p><strong>Artikulo 96 — Service Charges</strong> — Lahat ng manggagawa sa isang establishment na kumokolekta ng service charges ay may karapatan sa isang pantay o tamang bahagi sa 85% na kabuuang koleksiyon.</p>
                    <p><strong>Artikulo 95 — Service Incentive Leave</strong> — Bawat manggagawa na naglilingkod nang hindi kukulangin sa isang taon ay dapat magkaroon ng karapatan sa taunang service incentive leave ng limang araw na may bayad.</p>
                    <p><strong>R.A. 1161 — Maternity Leave</strong> — Ang bawat nagdadalang-taong manggagawa sa pribadong sektor, kasal man o hindi, ay makakatanggap ng maternity leave na katumbas ng humigit kumulang 100% na araw ng sahod.</p>
                    <p><strong>R.A. 8972 — Paternity Leave para sa mga solong magulang</strong> — Ipinagkaloob sa sinumang solong magulang o indibidwal na napag-iwanan ng responsibilidad ng pagiging magulang.</p>
                    <p><strong>R.A. 8187 — Paternity Leave</strong> — Maaaring magamit ng empleyadong lalaki sa unang apat na araw mula nang manganak ang legal na asawa na kaniyang kapisan. (Pakikipagpisan: obligasyon ng asawang babae at asawang lalaki na magsama sa isang bubong)</p>
                    <p><strong>Presidential Decree 851 — Thirteenth Month Pay</strong> — Nagbibigay sa mga empleyado nang hindi lalagpas ng ika-24 ng Disyembre bawat taon.</p>
                    <p><strong>Presidential Decree 626 — Benepisyo sa Employees Compensation Program</strong> — Isang compensation package sa mga manggagawa sa pampubliko at pambribadong sektor sakaling may kaganapang pagkakasakit na may kaugnayan sa trabaho.</p></div>
                    <div class="card"><div class="card-title">💡 Kahalagahan at Karagdagang Impormasyon</div>
                    <p>● Tinitiyak na makakarating sa mga mamimili ang mga produkto mula sa mga sakahan o pagawaan.</p>
                    <p>● Nagbibigay ng trabaho sa mga mamamayan. &nbsp;● Tinitiyak ang maayos na pag-iimbak at pagtitinda ng kalakal.</p>
                    <p>● Nagpapataas ng GDP ng bansa. &nbsp;● Nagpapasok ng dolyar sa bansa.</p>
                    <p><strong>Article 13, Section 3:</strong> Dapat magkaloob ang estado ng lubos na proteksyon sa paggawa, lokal at sa ibang bansa, at dapat itaguyod ang puspusang empleyo at pantay na mga pagkakataon sa trabaho para sa lahat.</p>
                    <p><strong>Karl Marx:</strong> ang mga manggagawa ang tunay na prodyuser ng bansa.</p>
                    <p><strong>Kontraktuwalisasyon:</strong> ang isang manggagawa ay nakatali sa kontrata ng 5 buwan lamang; kapag natapos ay kinakailangan nang umalis sa pinagtatrabahuhan.</p>
                    <p><strong>Brain Drain:</strong> pagkaubos ng mga manggagawa patungo sa ibang bansa. &nbsp;| <em>8 oras ang regular working hours.</em></p></div>

                    <div class="card card-gold"><div class="card-title">🏪 Lesson 5: Impormal na Sektor</div>
                    <p>Sektor ng ekonomiya na <strong>walang pormal o salat sa mga dokumentong kailangan</strong> sa pagsasagawa ng gawaing pang-ekonomiya. Ang kita ng sektor na ito ay <strong>HINDI</strong> naisasama sa kabuuang GDP ng bansa. Nagsisilbing paraan ng mamamayan upang magkaroon ng kabuhayan/dagdag kita sa tuwing panahon ng pangangailangan. Kilala rin bilang <strong>Underground Economy</strong>.</p></div>
                    <div class="card"><div class="card-title">👤 Halimbawa ng mga Taong Kabilang sa Sektor na Ito</div>
                    <p><strong>1.</strong> Nagtitinda sa kalsada &nbsp;<strong>2.</strong> Pedicab driver &nbsp;<strong>3.</strong> Karpintero &nbsp;<strong>4.</strong> Hindi rehistradong pampublikong sasakyan</p></div>
                    <div class="card"><div class="card-title">📋 Katangian ng Impormal na Sektor</div>
                    <p><strong>1.</strong> Hindi nakarehistro sa pamahalaan</p>
                    <p><strong>2.</strong> Hindi nagbabayad ng buwis mula sa kinita</p>
                    <p><strong>3.</strong> Walang legal at pormal na balangkas mula sa pamahalaan para sa pagnenegosyo</p></div>
                    <div class="card"><div class="card-title">🗂️ Iba't Ibang Anyo ng Impormal na Sektor</div>
                    <p><strong>1.</strong> Mga gawaing maaaring isakatuparan lamang sa loob ng tahanan</p>
                    <p><strong>2.</strong> Mga gawaing pansibiko, pagkakawanggawa, at panrelihiyon</p>
                    <p><strong>3.</strong> Mga sari-sari stores, pagtitinda sa labas ng bahay o sidewalk, paglalako ng kalakal o serbisyo</p>
                    <p><strong>4.</strong> Ilegal na gawain tulad ng pagnanakaw, pagbebenta ng ilegal na gamot, piracy, at prostitusyon</p></div>
                    <div class="card"><div class="card-title">💎 Kahalagahan ng Impormal na Sektor</div>
                    <p><strong>1.</strong> Sinasalo nito ang mga mamamayan na hindi makapasok bilang mga regular na empleyado sa isang kompanya.</p>
                    <p><strong>2.</strong> Nagbibigay ng pagkakataon sa maraming Pilipino na makapaghanapbuhay.</p>
                    <p><strong>3.</strong> Nagsisilbing tagasalo ng mga mamamayang may mahigpit na pangangailangan.</p>
                    <p><strong>4.</strong> Malaki ang naitutulong sa mga konsyumer ang mga kalakal at serbisyo mula sa impormal na sektor dahil sa MURA nitong halaga.</p></div>
                    <div class="card"><div class="card-title">❓ Dahilan kung Bakit Pumapasok sa Impormal na Sektor</div>
                    <p><strong>1.</strong> Makaligtas sa pagbabayad ng buwis sa pamahalaan</p>
                    <p><strong>2.</strong> Makaiwas sa masyadong mahaba at masalimuot na proseso ng pakikipagtransaksyon sa pamahalaan (bureaucratic red tape)</p>
                    <p><strong>3.</strong> Kawalan ng regulasyon mula sa pamahalaan na kung saan ang mga batas at programa ay hindi naitutupad nang maayos</p>
                    <p><strong>4.</strong> Makapaghanapbuhay nang hindi nangangailangan ng malaking kapital o puhunan</p>
                    <p><strong>5.</strong> Malabanan ang matinding kahirapan</p></div>
                    <div class="card"><div class="card-title">⚠️ Epekto sa Ekonomiya ng Impormal na Sektor</div>
                    <p><strong>1.</strong> Pagbaba ng halaga ng nalilikom na buwis</p>
                    <p><strong>2.</strong> Banta sa kapakanan ng mga mamimili</p>
                    <p><strong>3.</strong> Paglaganap ng mga ilegal na gawain</p></div>
                    <div class="card"><div class="card-title">📜 Mga Batas, Programa, at Patakaran sa Impormal na Sektor</div>
                    <p><strong>R.A. 8425 — Social Reform and Poverty Alleviation Act of 1997</strong> — Iahon sa kahirapan ang mga Pilipinong kabilang sa impormal na sektor. Itinatag ang National Anti-Poverty Commission.</p>
                    <p><strong>R.A. 9710 — Magna Carta of Women</strong> — Inaalis ang lahat ng uri ng diskriminasyon laban sa kababaihan.</p>
                    <p><strong>P.D. 422 — Philippine Labor Code</strong> — Pangunahing batas ng bansa para sa mga manggagawa na itinatag noong Mayo 1, 1974.</p>
                    <p><strong>R.A. 7796 — Technical Education and Skills Development Act of 1994</strong> — Layuning palakasin ang pakikiisa ng iba't ibang sektor upang mapabuti ang kasanayan at kalidad ng yamang tao sa bansa.</p>
                    <p><strong>R.A. 8282 — Social Security Act of 1997</strong> — Nagtatakda ng tungkulin ng estado na paunlarin at pangalagaan ang seguridad at kagalingang panlipunan ng mga manggagawa.</p>
                    <p><strong>R.A. 7875 — National Health Insurance Act of 1995</strong> — Nagtatag ng PhilHealth upang matiyak na may maayos at sistematikong kaseguruhang pangkalusugan ang lahat ng Pilipino.</p></div>
                    <div class="card"><div class="card-title">🤝 Mga Programa para sa Mamamayan</div>
                    <p><strong>DILP (Dole Integrated Livelihood Program)</strong> — Nagbibigay ng pagsasanay at tulong pangkabuhayan para sa sariling hanapbuhay.</p>
                    <p><strong>SEA-K (Self-Employment Assistance Kaunlaran Program)</strong> — Tumutulong sa mahihirap na pamilya sa pagsasanay at pagsisimula ng negosyo.</p>
                    <p><strong>ISLA (Integrated Services for Livelihood Advancement of the Fisherfolks)</strong> — Nagbibigay ng suporta sa pamamagitan ng pagsasanay at pagkakaloob ng kagamitan upang mapaunlad ang kanilang hanapbuhay.</p>
                    <p><strong>CWP (Cash-For-Work Program)</strong> — Pansamantalang kita para sa nasalanta, kapalit ng trabahong pang-rehabilitasyon.</p>
                    <p><strong>4Ps (Pantawid Pamilyang Pilipino Program)</strong> — Tulong-pinansyal sa pinakamahihirap na Pilipino kapalit ng pagsunod sa mga kondisyong pangkalusugan at pang-edukasyon ng kanilang mga anak.</p></div>

                <div class="ap-lesson" id="ap-l6">

const COOKIE_CONSENT_KEY = 'pacot_portfolio_consent';
const VISIT_COUNT_KEY = 'pacot_visit_count';

function setCookie(name, value, days) {
    const expires = new Date();
    expires.setTime(expires.getTime() + days * 24 * 60 * 60 * 1000);
    document.cookie = `${name}=${encodeURIComponent(value)};expires=${expires.toUTCString()};path=/;SameSite=Lax`;
}

function getCookie(name) {
    const cookies = document.cookie.split(';');
    for (let c of cookies) {
        const [k, v] = c.trim().split('=');
        if (k === name) return decodeURIComponent(v || '');
    }
    return null;
}

function deleteCookie(name) {
    document.cookie = `${name}=;expires=Thu, 01 Jan 1970 00:00:00 UTC;path=/`;
}

    const container = document.getElementById('petals-container');

}

        gateInfo.textContent = '🌸 First time here? Accept cookies to personalize your experience!';

function acceptCookies() {
    setCookie(COOKIE_CONSENT_KEY, 'accepted', 365);
    document.getElementById('cookie-banner').classList.remove('show');
    initPortfolioWithCookies();
}

function declineCookies() {
    setCookie(COOKIE_CONSENT_KEY, 'declined', 365);
    document.getElementById('cookie-banner').classList.remove('show');
    showToast('🌸 No worries! Some features may be limited.');
}

function initPortfolioWithCookies() {
    const consent = getCookie(COOKIE_CONSENT_KEY);
    if (consent !== 'accepted') return;

    // Visit counter
    let visitCount = parseInt(getCookie(VISIT_COUNT_KEY) || '0');
    const isFirstEver = !getCookie(FIRST_VISIT_KEY);

    if (isFirstEver) {
        setCookie(FIRST_VISIT_KEY, Date.now(), 365 * 5);
        setCookie(VISIT_COUNT_KEY, '1', 365);
        visitCount = 1;
    } else {
        visitCount += 1;
        setCookie(VISIT_COUNT_KEY, String(visitCount), 365);
    }

    // Update visit counter displays
    updateVisitDisplay(visitCount);

    // Welcome toasts
    const lastQuarter = getCookie(LAST_QUARTER_KEY);
    setTimeout(() => showWelcomeToast(visitCount, lastQuarter), 400);

    // Restore last quarter
    if (lastQuarter) {
        const qMatch = lastQuarter.match(/q1q2|q[3-4]/);
        if (qMatch) {
            const qtabs = document.querySelectorAll('.quarter-tab');
            qtabs.forEach(t => t.classList.remove('active'));
            const targetTab = [...qtabs].find(t => t.getAttribute('onclick')?.includes(qMatch[0]));
            if (targetTab) {
                targetTab.classList.add('active');
                showQuarter(qMatch[0], targetTab, false); // don't re-save
            }
        }
    }
}

function updateVisitDisplay(count) {
    const heroEl = document.getElementById('visit-counter-hero');
    const footerEl = document.getElementById('footer-visit-info');
    const msg = count === 1
        ? '🌸 First visit — welcome to my little world!'
        : `💗 You've visited ${count} time${count > 1 ? 's' : ''} — thank you for coming back!`;
    if (heroEl) heroEl.textContent = msg;
    if (footerEl) footerEl.textContent = `Visit #${count} — ${new Date().toLocaleDateString('en-PH', { month: 'long', day: 'numeric', year: 'numeric' })}`;
}

    document.querySelectorAll('.quarter-pane').forEach(p => p.classList.remove('active'));

    const s = document.createElement('span');

function spawnSparkle(el) {
